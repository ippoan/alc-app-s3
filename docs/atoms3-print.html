alc-hub AtomS3 Print

AtomS3 + Atomic PoE Base 印刷ブリッジ ファームウェア書き込みツール — 点呼記録 PDF を営業所プリンター (9100/raw) へ自動印刷します。 CoreS3 統合ハブは こちら。

対応機器

マイコンM5Stack AtomS3 (SKU C123)
ネットワークAtomic PoE Base (SKU A091, W5500 + PoE 給電)
プリンター9100/tcp (raw) 対応・PDF ダイレクトプリント対応機

書き込み手順

  1. AtomS3 を USB-C ケーブルで PC に接続 (PoE Base は外れていてもよい)
  2. 下の「Install」ボタンをクリックし、シリアルポートを選択
  3. 書き込み完了後、PoE Base に載せて LAN ケーブル (PoE) を接続
  4. 画面に ETH UP と IP アドレスが出れば疎通完了

動作要件

  • Chrome / Edge / Opera (Web Serial API 対応ブラウザ)
  • AtomS3 は USB Serial/JTAG 内蔵のため追加ドライバ不要
  • 認識されない場合は本体側面のリセットボタンを約 2 秒長押しして書き込みモードへ

ファームウェアをインストール

お使いのブラウザは Web Serial API に対応していません。
Chrome, Edge, または Opera をお使いください。 シリアルポートへのアクセスが許可されていません。

初期設定 (シリアル 115200bps)

書き込み後の設定はシリアルコマンドで行います (デバイス登録は auth.ippoan.org/device/setup、 印刷テストは /device/print-test からも実行可)。

  • PRINTER ADDR 192.168.x.x:9100 — プリンター宛先の保存 (NVS)
  • PRINT https://…/test.pdf — 手動テスト印刷 (進捗は EVT PRINT_*)
  • AUTH SET <id> <secret> <tenant> → AUTH TOKEN — デバイス登録と疎通確認
  • WS STATUS — 遠隔管理 (下り print/ota) 接続の確認
  • OTA <url> — 手動ファームウェア更新
