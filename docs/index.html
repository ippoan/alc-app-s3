alc-hub CoreS3

M5Stack CoreS3 用 点呼キオスク統合ハブ ファームウェア書き込みツール

対応機器

マイコン M5Stack CoreS3
アルコールチェッカー FC-1200 (RS232M Module 経由)
体温計 ニプロ NT-100B (BLE)
血圧計 ニプロ NBP-1BLE (BLE)

書き込み手順

  1. CoreS3 を USB-C ケーブルで PC に接続
  2. 下の「Install」ボタンをクリック
  3. シリアルポートを選択（COM○ または /dev/ttyACM○）
  4. 書き込み完了を待つ（1〜2分）

動作要件

  • Chrome, Edge, Opera のいずれか（Web Serial API 対応ブラウザ）
  • CoreS3 は USB Serial/JTAG 内蔵のため、通常は追加ドライバ不要（Windows 10 以降 / macOS / Linux）

注意事項

  • 書き込み中は USB ケーブルを抜かないでください
  • ポートが認識されない・書き込みが始まらない場合は、リセットボタンを約2秒長押しして書き込みモード（ダウンロードモード）に入れてから再試行してください
  • 書き込み後、CoreS3 は自動的に再起動し、起動画面 → 待機画面（アルコールチェック）が表示されます

ファームウェアをインストール

お使いのブラウザは Web Serial API に対応していません。
Chrome, Edge, または Opera をお使いください。 シリアルポートへのアクセスが許可されていません。

書き込み後の動作確認

  • 画面: 起動画面の後、待機画面「アルコールチェック」とステータスバー (LAN / 232 / BLE) が表示されます
  • タッチ: 待機画面をタップすると機器ステータス詳細が表示されます
  • シリアル (115200bps): PING 送信 → PONG 応答、QR https://example.com 30 で QR 表示をテストできます
  • BLE: 体温計/血圧計で測定すると {"type":"temperature",...} 形式の JSON が出力されます (ble-medical-gateway 互換)
